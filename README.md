## REST API Reference
| Return Type   | HTTP Method   | URI               | Request Body  | Purpose
|-------------  |---------------|-------------------|---------------|
| List<Game>    | GET           | /api/games        |               | List
| Game          | GET           | /api/games/{id}   |               | Retrieve

| Void          | DELETE        | /api/games/{id}   | Game JSON     | Delete
| Game          | PUT           | /api/games/{id}   | Game JSON     | Update
